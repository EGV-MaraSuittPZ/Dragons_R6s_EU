# Dragons_R6s_EU
Web oficial de DRAGONS, equipo de Esports de Rainbow Six Siege. Diseñada con una estética táctica premium y oscura, esta plataforma presenta nuestra alineación activa de jugadores, el staff técnico, los últimos resultados de nuestros partidos en ataque y defensa, nuestro palmarés competitivo, la historia del club y canales de contacto oficial.
